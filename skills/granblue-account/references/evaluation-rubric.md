# Evaluation Rubric

## What To Capture

For each element:

- Main party and standout substitutes
- Main summon, support summon preference, and strongest subs
- Weapon grid theme and missing anchor pieces
- Any visible progression flags: rank, uncaps, EMPs, event readiness, sandbox/readiness notes
- One immediate action and one medium-term action

## How To Judge Account Value

Use a rough, practical value estimate rather than pretending to have market precision.

- `Early`
  - Roster and grids are still broad but shallow.
  - Advice should focus on one or two carry elements and core account systems.
- `Mid`
  - Several usable teams exist.
  - Advice should focus on finishing one reliable farm setup and fixing the weakest bottlenecks.
- `Advanced`
  - Multiple elements are clearly functional for harder content.
  - Advice should focus on coverage, consistency, and premium upgrade efficiency.
- `High-end`
  - The account can pivot between multiple endgame teams with strong summon depth.
  - Advice should focus on marginal gains, niche coverage, and resource optimization.

## Priority Order For Recommendations

1. Remove blockers that affect every element: classes, summons, core systems, inventory friction, event participation.
2. Finish the most complete element first instead of spreading upgrades evenly.
3. Build around strong summons and modern core units already owned.
4. Prefer upgrades that unlock repeatable farming over vanity upgrades.
5. Call out when a weaker element should be parked temporarily.
