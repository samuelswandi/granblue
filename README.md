# Granblue Resource Tracking

This repo is for read-only Granblue Fantasy resource tracking.

Scope:

- track character, weapon, summon, and account progress
- record screenshots-derived observations
- maintain local docs for roster evaluation and progression planning
- recommend next upgrades and priorities

Non-scope:

- gameplay automation
- farming automation
- combat scripting
- botting or unattended play

Primary files:

- `docs/account-state.md`
- `skills/granblue-account/`

Working rule:

- prefer evidence from the user's live session, screenshots, and explicit statements
- treat account value and upgrade advice as judgment, not exact market pricing
